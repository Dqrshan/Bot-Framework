# Discord.js v14 Bot Framework

A framework for creating Discord Bots in **typescript**!

---

## Instructions

### Prerequisite
- [Node.js]((https://nodejs.org/)) v16+
```bash
npm install
```

### Running the code

```bash
npm run watch:start
```
